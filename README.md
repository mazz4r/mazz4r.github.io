# mazz4r.github.io

Personal resume website built as a weekend project — now a living document of my academic and professional journey.
About
A fully static, single-file resume site built with pure HTML, CSS, and vanilla JavaScript. No frameworks, no dependencies, no build step — just one index.html you can open in any browser or host anywhere.
Designed with a dark editorial aesthetic, the site features 7 interactive tabs covering experience, extracurricular activities, honors, skills, and more — all with smooth animations and a custom cursor.
Stack

HTML5 + CSS3 — layout, animations, transitions
Vanilla JS — tab navigation, animated skill bars, custom cursor
Google Fonts — Cormorant Garamond, DM Mono, Instrument Sans
No frameworks. No build tools. No dependencies.

Features

Sidebar navigation with 7 content sections
Animated timeline for experience entries
Skill bars with level indicators
Language cards with flags
Responsive layout with mobile hamburger menu
Custom animated cursor (desktop)
Print / PDF export support

Structure
mazz4r.github.io/
├── index.html     # entire site — styles, markup, and scripts in one file
├── .nojekyll      # disables Jekyll so GitHub Pages serves static files directly
└── README.md
